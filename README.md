# pocsag-gateway

This WIP Software aims to provide an Interface for transmitting Messages sent over AMQP via an RFM69 Transceiver.

## Pin Configuration
| RFM pin | Pi pin  
| ------- |-------
| DIO0    | 18 (GPIO24)  
| MOSI    | 19  
| MISO    | 21  
| CLK     | 23  
| NSS     | 24  
| Ground  | 25  
| RESET   | 29
